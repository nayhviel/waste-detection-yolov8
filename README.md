# Recyclable Waste Detection using YOLOv8s

This project implements recyclable waste detection using YOLOv8s on the WaRP Dataset.

Classes:
- canister
- cardboard
- bottle
- detergent
- cans

Models:
- YOLOv8s Baseline
- YOLOv8s + Lighting Augmentation

Developed for research:
"Optimasi Kinerja YOLOv8 melalui Augmentasi Pencahayaan untuk Pengenalan Sampah Daur Ulang"
